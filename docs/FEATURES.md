# مميزات منصة وثبة التعليمية

هذا الملف يوثّق كل ميزة موجودة في منصة **وثبة** بشكل مفصّل: عنوان الميزة ووصف مختصر تحته.

---

## أولاً: مميزات عامة للمنصة

### نظام Multi-Tenant (تعدد المنصات)
كل مدرس يحصل على منصة مستقلة برابط (Subdomain/Slug) خاص به مع عزل تام لبياناته عن باقي المدرسين.

### الصفحة الترويجية للمنصة (PlatformHome)
صفحة عامة على النطاق الرئيسي تعرّف بخدمة وثبة كمنتج SaaS لمدرسي السناتر.

### صفحة هبوط المدرس (Landing Page)
واجهة عامة لكل منصة مدرس تعرض هويته البصرية ونبذة عنه قبل تسجيل الدخول.

### تسجيل دخول موحّد بثلاثة أدوار
شاشة دخول واحدة تتعرّف تلقائياً على نوع المستخدم (معلم / مساعد / طالب) وتوجّهه للوحته.

### مصادقة JWT آمنة
كل الجلسات والطلبات محمية بـ JWT مع تشفير كلمات المرور عبر bcrypt.

### الوضع الليلي والنهاري (Dark/Light Mode)
إمكانية التبديل بين الثيم الفاتح والداكن مع حفظ تفضيل المستخدم.

### تطبيق ويب تقدّمي (PWA)
يمكن تثبيت المنصة على الموبايل كتطبيق مع Service Worker وبانر تثبيت مخصص.

### سياسة الخصوصية والشروط والأحكام
صفحات قانونية جاهزة (Privacy Policy و Terms) متاحة للمستخدمين.

### تصميم متجاوب بالكامل (RTL)
واجهة عربية تعمل بسلاسة على الموبايل والتابلت والديسكتوب بدعم كامل للكتابة من اليمين لليسار.

## ثانياً: مميزات المعلم (Teacher)

### لوحة تحكم المعلم (Dashboard)
ملخص سريع لأهم الإحصائيات: عدد الطلاب، الكورسات، الامتحانات، والإيرادات.

### إدارة الكورسات (Courses)
إنشاء وتعديل وحذف الكورسات مع تحديد السعر والمرحلة الدراسية والصورة المصغّرة.

### إدارة محتوى الكورس (Course Content)
تنظيم الكورس في أقسام ورفع فيديوهات وملفات PDF مرتبطة بكل قسم.

### إدارة الطلاب (Students)
إضافة وتعديل وحذف الطلاب مع بحث وفلترة واستيراد جماعي وتصدير القوائم.

### بروفايل الطالب التفصيلي (Student Profile)
عرض تقدّم الطالب في الكورسات ودرجاته وشاراته ومنحنى أدائه عبر الزمن.

### إدارة المساعدين (Assistants)
إنشاء حسابات مساعدين والتحكم في 9 صلاحيات منفصلة لكل مساعد.

### إدارة الامتحانات (Exams)
إنشاء امتحانات بأنواع أسئلة متعددة (MCQ / صح وخطأ / مقالي) مع إعدادات متقدمة.

### محرر أسئلة الامتحان (Exam Questions)
إضافة وتعديل أسئلة الامتحان مع دعم المعادلات الرياضية والصور.

### بنوك الأسئلة (Question Banks)
إنشاء بنوك أسئلة قابلة لإعادة الاستخدام وسحب أسئلة عشوائية لكل طالب.

### الأسئلة الأكثر خطأً (Wrong Questions)
تقرير يبرز الأسئلة التي يخطئ فيها معظم الطلاب لتحديد نقاط الضعف.

### طلبات إعادة الامتحان (Retry Requests)
مراجعة طلبات الطلاب لإعادة الامتحان والموافقة أو الرفض مع ملاحظة.

### مراجعة نتيجة الامتحان (Exam Review)
استعراض إجابات الطالب سؤالاً بسؤال وتصحيح الأسئلة المقالية يدوياً.

### نظام التسميع (Recitations)
إنشاء جلسات تسميع وأسئلة خاصة بها (مفيد لحفظ القرآن أو النصوص) ومراجعتها.

### أسئلة التسميع (Recitation Questions)
إدارة أسئلة كل جلسة تسميع ومراجعة أداء الطلاب فيها.

### إدارة المدفوعات (Payments)
مراجعة إيصالات الدفع والتحقق منها (InstaPay / Vodafone Cash / Fawry) وتتبع الإيرادات.

### طلبات التسجيل في الكورسات (Requests)
مراجعة طلبات الطلاب للانضمام للكورسات والموافقة أو الرفض.

### الحضور والغياب (Attendance)
تسجيل ومتابعة حضور الطلاب للجلسات والكورسات.

### التحليلات (Analytics)
رسوم بيانية تفصيلية لأداء الطلاب ومتوسط الدرجات ونسب النجاح لكل كورس.

### لوحة المتصدرين (Leaderboard)
ترتيب الطلاب حسب النقاط مع إمكانية إعادة الضبط الدوري وحفظ السجل التاريخي.

### الإشعارات (Notifications)
إرسال إشعارات يدوية للطلاب وأولياء الأمور مع اختيار المستلمين.

### البث المباشر (Live Stream)
إنشاء جلسات بث مباشر بتقنية LiveKit مع شات ورفع يد وإدارة المتحدثين.

### سجل النشاط (Activity Log)
سجل بكل العمليات الحساسة التي تتم على المنصة لأغراض المتابعة والمراجعة.

### النسخ الاحتياطي (Backup)
تصدير واستيراد بيانات المنصة للحفاظ عليها أو نقلها.

### الأرشيف (Archive)
أرشفة الطلاب والبيانات القديمة مع إمكانية الاستعادة لاحقاً.

### إعدادات المنصة (Settings)
تخصيص اسم المنصة والشعار والصورة الشخصية وبيانات التواصل وإعدادات النقاط.

### تكامل واتساب (WhatsApp)
التواصل المباشر مع الطلاب وأولياء الأمور عبر روابط واتساب.

## ثالثاً: مميزات المساعد (Assistant)

### لوحة تحكم المساعد (Dashboard)
واجهة مستقلة تعرض فقط الأقسام المسموح بها حسب صلاحيات المساعد.

### نظام الصلاحيات المتدرّج (9 صلاحيات)
المعلم يفعّل أو يعطّل كل صلاحية على حدة: إدارة الطلاب، الكورسات، الامتحانات، التسميع، المدفوعات، التحليلات، الإشعارات وغيرها.

### وصول مشروط للصفحات (Permission Routes)
كل صفحة للمساعد محمية ببوابة صلاحية تمنع الدخول إن لم تكن الصلاحية مفعّلة.

---

## رابعاً: مميزات الطالب (Student)

### لوحة تحكم الطالب (Dashboard)
عرض النقاط والترتيب والكورسات المسجّلة والامتحانات القادمة وآخر نشاط.

### تصفّح الكورسات (Courses)
استعراض الكورسات المتاحة والمسجّل فيها مع نسبة الإكمال.

### مشاهدة الكورس (Course View)
مشاهدة الفيديوهات وتحميل ملفات PDF مع حفظ تلقائي لتقدّم المشاهدة.

### تأدية الامتحانات (Exams)
أداء الامتحانات المتاحة بمؤقت محسوب على السيرفر مع تصحيح تلقائي فوري.

### مراجعة الامتحان (Exam Review)
استعراض الإجابات الصحيحة والخاطئة بعد تسليم الامتحان.

### التسميع (Recitations)
أداء جلسات التسميع المخصصة ومراجعة نتائجها.

### إحصائياتي (My Stats)
عرض منحنى الدرجات الشخصي والشارات والمقارنة مع متوسط المجموعة.

### لوحة المتصدرين (Leaderboard)
متابعة ترتيب الطالب بين زملائه ونقاطه.

### الإشعارات (Notifications)
استقبال الإشعارات الفورية داخل المنصة وعبر Push Notifications.

### البث المباشر (Live Stream)
حضور جلسات البث المباشر والمشاركة في الشات ورفع اليد.

### الفعاليات الأسبوعية (Events)
المشاركة في الفعاليات التفاعلية الأسبوعية لكسب نقاط إضافية.

### لعبة Stickman Run
لعبة تعليمية بالـ Canvas فيها قفز وتحاشي عقبات ومواجهات Boss بأسئلة حسب المرحلة الدراسية.

### بوابة ولي الأمر (Parent Portal)
صفحة مخصصة لولي الأمر لمتابعة أداء ابنه وحضوره ونتائجه.

## خامساً: مميزات تقنية وأمنية

### تتبّع تقدم الفيديو (Video Progress)
حفظ الدقائق المشاهَدة والنسبة المئوية وآخر موضع تشغيل وعدد مرات المشاهدة لكل فيديو.

### التحديثات اللحظية (SSE)
دفع التحديثات الفورية للعميل عبر Server-Sent Events (نشر امتحان، طلب إعادة، بدء بث).

### إشعارات Push (FCM)
إرسال إشعارات للموبايل عبر Firebase Cloud Messaging حتى خارج المنصة.

### حماية ضد التقاط الشاشة (Anti-Capture)
آلية لتقليل إمكانية تصوير أو التقاط محتوى الفيديوهات المحمية.

### دعم المعادلات الرياضية (MathText)
عرض المعادلات الرياضية داخل الأسئلة والمحتوى بشكل صحيح.

### حماية الملفات
الفيديوهات وملفات PDF لا يمكن الوصول إليها إلا بتوكن JWT صالح.

### Rate Limiting وحماية الطلبات
تحديد عدد الطلبات لكل مستخدم لمنع إساءة الاستخدام مع HTTP Security Headers عبر Helmet.

### Cache صلاحيات المساعدين
تخزين مؤقت لصلاحيات المساعدين لتسريع التحقق وتقليل الضغط على قاعدة البيانات.
